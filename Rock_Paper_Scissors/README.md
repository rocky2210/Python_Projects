# Simple Rock Paper Scissors Project

This is a simple command-line Rock paper scissors project written in Python. It allows users to play their childhood favourite game

## Features

- Infinity gameplay
- Scoreboard.
- Simple colorful interface.
- Exit option.

### Prerequisites

- Python 3.x

### Usage

1. Run the rock paper scissors by executing the `main.py` file:

   ```shell
   python3 main.py
   ```

2. You will see the menu with the available options:

   ```
    User commands
     1. Play a round
     2. View scoreboard
     3. Exit
   ```

3. Enter the number corresponding to your desired operation:

   - For rock, enter `1`.
   - For scissors, enter `2`.
   - For paper, enter `3`.
