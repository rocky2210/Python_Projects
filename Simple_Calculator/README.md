# Simple Calculator Project

This is a simple command-line calculator project written in Python. It allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division. The calculator also provides a previous result feature, allowing users to use the result of the last operation in a new calculation.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Previous result feature
- Error handling for invalid inputs

### Prerequisites

- Python 3.x

### Usage

1. Run the calculator by executing the `calculator.py` file:

   ```shell
   python3 calculator.py
   ```

2. The calculator will display a menu with options for different operations.

3. Choose an operation by entering the corresponding number (1 to 6) and follow the prompts to enter the required numbers.

4. The result of the operation will be displayed along with a visual representation of the result in a box.

5. You can exit the calculator by choosing option 6.
